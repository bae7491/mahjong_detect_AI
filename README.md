# mahjong_detect_AI
## 개요 (프로젝트 소개)
### 일본 리치마작을 오프라인에서 진행할 때 마작 초보들은 제대로 점수를 계산하지 못한다. 따라서 초보들을 위해 화료이후 사진을 찍으면 그 사진을 인식하여 점수를 계산해주는 프로그램이다.
